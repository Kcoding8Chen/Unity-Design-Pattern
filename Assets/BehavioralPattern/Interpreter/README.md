# 解释器模式
![解释器](/images/2023/02/14/%E8%A7%A3%E9%87%8A%E5%99%A8.png)

> **解释器模式**：给定一个语言，定义它的文法的一种表示，并定义一个解释器，这个解释器使用该表示来解释语言中的句子。

## 优缺点

**主要缺点：**

- 由于有大量的循环和递归调用，从而导致运行效率很低。
- 复杂文本难以维护。

**优点：**

- 增加新的表达式较为方便
- 单独改一个文法较为容易。
